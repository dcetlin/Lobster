# Lobster Knowledge Vault

Welcome to your Obsidian vault, managed by Lobster.

## Folder Structure

| Folder    | Purpose                                                |
|-----------|--------------------------------------------------------|
| `Inbox/`  | Quick captures and unprocessed notes from Lobster      |
| `Notes/`  | Permanent notes and knowledge base                     |
| `Links/`  | Archived web links with context                        |
| `Daily/`  | Daily notes and journal entries                        |
| `Archive/`| Inactive notes and completed projects                  |

## How Lobster Uses This Vault

Lobster can read and write notes directly to this vault via the Obsidian KM skill.

**Default behaviors:**
- Quick notes go to `Inbox/` for later processing
- Links captured via conversation go to `Links/`
- Daily digests go to `Daily/YYYY-MM-DD.md`

## Syncing

This vault syncs between devices via CouchDB (PouchDB protocol). Your Obsidian clients connect directly to the sync server — no third-party cloud required.

## Getting Started

1. Open this folder in Obsidian
2. Connect your sync plugin to the CouchDB server
3. Ask Lobster to take a note!
