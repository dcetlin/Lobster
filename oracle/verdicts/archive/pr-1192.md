- Sonnet tier: `lobster-generalist`
- Haiku tier: `session-note-appender`

The `session-note-appender.md` agent definition has `model: claude-haiku-4-5-20251001` in its frontmatter. The fix is correct and complete for its stated scope.

The scope decision is correct.

This is NOT an Encoded Orientation decision. It corrects a pre-existing dispatch site to use the already-defined correct agent type. No new behavioral default is being established; no new decision-making rule is being added. The `session-note-appender` agent definition pre-existed this PR. The fix repairs a mismatch between the dispatch instruction and the intended agent, not a design change. Constraint-3 does not apply.

**Failure modes:** None introduced. The `session-note-appender` agent definition exists and is correctly defined. Reverting this change would reinstate the cost regression; keeping it forward restores correct model routing.

**Patterns introduced:** None new. This is a one-line correction.
