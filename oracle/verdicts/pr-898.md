## Stage 2: Quality Review

- **Self-check guard**: Correct scope and pattern. `grep -qE '^\s*(python3|python|pip)\s' "$0"` fires on line-leading bare invocations only, exempting jq heredoc strings. The guard runs before any migration — fail-fast, no partial migration. Side-effect free when clean.

- **`uv venv` vs `python3 -m venv`**: Functionally equivalent. `uv venv` creates a compatible venv, is faster, and is the correct uv idiom. Downstream `source activate` and `bin/python` calls are unaffected.

- **`$VENV_DIR/bin/python` for playwright calls**: Sound. The venv is activated in the same function before these calls, so bare `python` would work — but the explicit path is more defensive and consistent with the existing pattern at line 1306 (as noted in the PR description). The PR description correctly identified this as "consistent with existing pattern."

- **`record-catchup-state.sh` stale comment removal**: Correct. The comment "Uses python3 (not uv) because this script runs outside the Claude/uv context" was incorrect — uv is available system-wide. Removing it is the right action.

- **Completeness**: Author ran `grep -rn '^\s*(python3|python|pip)\s' scripts/*.sh` post-fix and got no output. Diff confirms all 6 scripts are accounted for in the PR description. No feature bundling — 6 files changed, all scripts/*.sh, all named in the description.

- **Minor observation**: `uv pip install --quiet --upgrade pip` (upgrading pip via uv) is a mild conceptual redundancy — with uv, pip upgrading is less critical since uv manages resolution natively. Not a correctness issue; harmless.

**Patterns introduced**: The upgrade.sh self-check guard establishes a structural enforcement pattern for shell-script conventions. This is reusable: any convention that applies to a shell script can be enforced with a similar guard that grep-scans the script's own contents before running. This is the "advisory-to-hook promotion" pattern from golden-patterns.md applied at the shell level rather than the Claude Code hook level.

**What this forecloses**: Nothing. The uv convention was already in CLAUDE.md. This PR makes existing scripts conform and adds a structural guard that prevents new violations in upgrade.sh specifically.

**Opportunity cost**: Low. This is a bounded sweep with no architectural tradeoffs.
